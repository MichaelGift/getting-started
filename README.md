# User-Stories
- As a student, I want to Report a Missing Item so that you can find your belonging.
- As Person Found i will take to lost and found office.
- as staff will post picture on website as lost item.
- As Lost item person i will go to website and check for my belongings if i see my item i will go to lost and found office.
- As a staff will reported found or resolved the missing item or gets deleted.


## User Personas:
1. Student: Either a student who has lost an item or found an item and wants to report it.
2. Staff: Responsible for managing the lost and found items, posting them on the website.


## User story 1: Report a Missing Item
 - Student Fills A Form With Details Of The Missing Item
 - Server Receives The Form Data And Stores It In A Database
 - Server Sends A Confirmation Message To The User

 ## User story 2: Update the Form
 - Staff looks for the report and update the status as Available.
 - Server send out notification text to the users.
 
 ## User story 3: Claim for Missing Item
 - Student Claims for lost item.
 - Staff gets the items and update the status as Found.
 - server send out notification text Resolved.
